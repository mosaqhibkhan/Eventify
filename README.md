# Eventify – Crowd Verified Campus Event Platform

Eventify is a platform where students share and verify information about college events such as fests, graduations, fashion shows, and trips.

Businesses like cafes, hotels, restaurants, and clothing stores can attach event-based discounts to attract students attending those events.

## Live Demo

Frontend: https://eventify-seven-murex.vercel.app
Backend API: https://eventify-backend-1i56.onrender.com

## Features

* Crowd-verified campus event information
* Reward system for students who report events
* Business marketplace to attach targeted offers
* Event discovery platform for students
* Leaderboard for top contributors
* Trending events based on business offers

## How It Works

1. Students submit upcoming college events.
2. The first verified reporter earns reward points.
3. Businesses attach offers related to those events.
4. Students discover events and available discounts.
