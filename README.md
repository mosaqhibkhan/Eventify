# Eventify – Crowd Verified Campus Event Platform

Eventify is a platform where students share and verify information about
college events such as fests, graduations, fashion shows, and trips.

Businesses like cafes, hotels, restaurants, and clothing stores can attach
event-based discounts to attract students.

## Features

- Crowd verified event information
- Reward system for contributors
- Business marketplace for offers
- Event discovery platform for students
